# Critical Findings

Append sanitized, durable findings here when a discovery may affect future jobs.

Do not include secrets, private keys, seed phrases, API keys, operator personal
information, or unnecessary client personal information.

## Template

```md
### YYYY-MM-DD - <jobId> - <severity> - <finding type>

- Evidence:
- Impact:
- Action taken:
- Durable lesson:
- Reuse trigger:
```
